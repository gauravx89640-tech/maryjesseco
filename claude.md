# Forge Studio — Business Context

## Who I Am
Gaurav, based in Indore, Madhya Pradesh, India. I run a freelance web design business called Forge Studio targeting small US businesses (cafés, bakeries, gyms, barbershops, nail salons, beauty salons, local service businesses). Zero coding knowledge — I act as designer and project manager between client and AI.

## Business Model
Find small US businesses with no website → cold email → offer free mock-up → build with v0 by Vercel (Next.js) → host on Cloudflare Pages → deliver and collect payment.

## Tech Stack
- Next.js via v0 by Vercel
- Hosted on Cloudflare Pages
- GitHub repo → Cloudflare auto-deploys
- Images must be local (not Vercel blob URLs)
- Web3Forms for contact forms
- GitHub browser editor for edits post-delivery

## Cold Outreach
- Primary: gaurav@forgestudio.one — all cold outreach
- Secondary: Gmail (builtwithforge@ or similar) — additional volume
- Replies and client communication: hello@forgestudio.one only
- Volume: 50 touchpoints/day (cold + follow ups)
- Sending window: 7:30-9:30pm IST
- Heaviest volume: Monday-Wednesday
- Follow up 1: day 4-5, Follow up 2: day 8-10
- Spread FU dates across 5 days per batch (8 per day max)
- After 2 follow ups no reply → mark cold

## Mock-up to Close Process
1. Prospect says yes to mock-up
2. Reply: "Perfect — do you have an existing website or domain I should know about? I'll have something ready by [2 days out]"
3. Build immediately using Maps listing for details
4. Send: "Built this for you — take a look and tell me what you think"
5. Wait for reaction → then proposal
6. Exception: very warm prospects (sending photos etc) → preview + proposal same day
7. Never send questionnaire before mock-up

## Care Plan Pitch

- Frame as removing mental baggage: "you just text me and it's handled"
